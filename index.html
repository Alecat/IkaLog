読んでね / Must Read

WinIkaLog を目的の言語向けで起動させるために、以下に示すバッチファイルを使用してください。
Use the batch file below to Run WinIkaLog in your region mode.

日本語版 (Japanese)WinIkaLog_ja.BAT
北米版 (North America)WinIkaLog_NA.BAT
ヨーロッパ版 (EU)WinIkaLog_EU.BAT

MSVCRT100.DLL がなくて IkaLog が起動しない場合は次のランタイムをインストールしてください。
If you don't have MSVCRT100.DLL, try install the runtime below.

Microsoft Visual C++ 2010 Redistributable Package (x86) (日本語; Japanese)
Microsoft Visual C++ 2010 Redistributable Package (x86) (英語; US English)

最近のバージョンでアマレコからの入力がうまくいかないケースがあるようです。この場合でも、入力方法を DirectShow に設定し、 AmaRec Video Capture を手動選択すると動くようです。
